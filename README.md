## About

ARIELA is a bilingual (French / English) mobile application for iOS and Android focused on women's health across every life stage. It goes beyond a simple period tracker by offering personalized education, AI-powered guidance, and a safe, private experience.

Built with a strong focus on privacy, accessibility, and inclusivity — designed to serve women in Europe and French-speaking Africa equally.

## Key features

- **First Period Mode** — Educational, reassuring content for young girls
- **Cycle Tracker** — Calendar, predictions, symptom tracking
- **Safe Mode** — Fertility window awareness for contraception planning
- **Fertility Mode** *(Premium)* — Conception support with advanced insights
- **Pregnancy Mode** *(Premium)* — Week-by-week pregnancy tracking
- **Postpartum Mode** *(Premium)* — Recovery, breastfeeding, mental health
- **AI Health Assistant** *(Premium)* — Personalized guidance powered by GPT-4o

## Tech stack

| Layer | Technology |
|---|---|
| Mobile framework | Flutter (Dart) |
| State management | Riverpod |
| Local database | Hive |
| Backend | Supabase (PostgreSQL + Auth + Edge Functions) |
| Authentication | Email + Google + Apple Sign In |
| AI | OpenAI API (GPT-4o-mini + GPT-4o) |
| Subscriptions | RevenueCat |
| Push notifications | Firebase Cloud Messaging |
| Internationalization | flutter_localizations (FR + EN) |

## Privacy & medical disclaimer

ARIELA handles sensitive health data with the highest privacy standards. The application is **not a medical device** and does not provide medical diagnosis. The AI assistant gives general information only and always recommends consulting a qualified healthcare professional for any health concern. See [AI Disclaimer](./docs/ai-disclaimer.md) for details.
